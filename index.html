⚡ Connecter Firebase

Suivez ces 5 étapes — gratuit, 5 minutes, synchronisation instantanée entre toutes vos boutiques

1
Allez sur console.firebase.google.com avec votre compte Google → Créer un projet → donnez un nom (ex : ma-boutique-dakar) → désactiver Google Analytics → Créer
2
Dans le menu gauche → Build → Firestore Database → Créer une base de données → choisir Mode production → sélectionner eur3 (Europe) ou nam5 → Activer
!
Règles de sécurité Firestore — Dans Firestore → onglet Règles → remplacer tout par :
rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /users/{uid}/{document=**} {
      allow read, write: if request.auth != null && request.auth.uid == uid;
    }
  }
} → Publier
3
Menu gauche → Build → Authentication → Commencer → onglet Sign-in method → activer Google (+ votre email support) → activer E-mail/Mot de passe → Enregistrer
4
⚙ Icône engrenage → Paramètres du projet → section Vos applications → cliquer </> (Web) → donner un nom → Enregistrer l'application → copier tout le bloc firebaseConfig = { ... }
5
Collez la configuration dans le champ ci-dessous et cliquez Enregistrer
